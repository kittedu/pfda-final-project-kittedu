# Bullet Dodging Game

## Repository
https://github.com/kittedu/pfda-final-project-kittedu.git

## Description
For my final project, I decided to make a bullet dodging game that implements collision and movement. To give this project a unique look, I'll include some of my own custom art assets.
This is relevant to the digital arts because not only does it involve the technical aspects of problem solving with a digital program, the tools that I implement will also be able to contain and display art to the player.

## Features
- Movement
	- By using directional keys, the player can move and dodge bullets. 	
- Death
	- If a bullet is touched, the player dies.
- Levels
	- If the player survives for a certain amount of time, they pass to the next level.
- Bullets
	- Bullets will fly across the screen for the player to avoid.
- Progression
	- As the levels increase, the bullets speed up and/or there will be more bullets.     
- Art Assets 
	- The objects in the game will feature custom art that I made.

## Challenges
- I need to understand how to implement collision for all moving objects. 
- I need to make use of counting the ticks/time that the player spends alive in order to determine whether to go on to the next level.
- There needs to be a variable that contains the 'difficulty' of the level in order to scale up the speed or amount of bullets.
- I need to incorporate a system that creates new bullets if the difficulty increases.
- I need to code the program to read and interpret directional key input.

## Outcomes
Ideal Outcome:
- The ideal outcome here is that I create a fully functional game with all of the features above that is aesthetically pleasing.
- Additional elements that improve the experience are included like a counter for how many levels have been passed and a high score system.

Minimal Viable Outcome:
- At the bare minimum, the core game mechanics are implemented. The mechanics include the bullets, death, and levels.
- If I run out of time for something, I should prioritize coding the games mechanics over creating and implementing art assests because its a programming project.
